# Πλειστηριασμοί Κύπρου — Οδηγίες στησίματος (15 λεπτά, χωρίς εγκαταστάσεις)

Όλα γίνονται μέσα από τον browser. Δεν χρειάζεσαι Python, Command Prompt, τίποτα.

## Βήμα 1 — Λογαριασμός GitHub (αν δεν έχεις)

1. Πήγαινε στο github.com → Sign up
2. Email, κωδικός, όνομα χρήστη — 2 λεπτά

## Βήμα 2 — Δημιουργία repository

1. Πάνω δεξιά πάτα το **+** → **New repository**
2. Όνομα: `auctions-cy`
3. Επίλεξε **Public** (απαραίτητο για δωρεάν φιλοξενία)
4. Πάτα **Create repository**

## Βήμα 3 — Ανέβασμα αρχείων

1. Στη σελίδα του repository πάτα **uploading an existing file** (ή Add file → Upload files)
2. Σύρε μέσα τα αρχεία: `scraper.py`, `index.html`, `requirements.txt`, `ΟΔΗΓΙΕΣ.md`
3. Πάτα **Commit changes** (πράσινο κουμπί κάτω)

## Βήμα 4 — Το αρχείο αυτοματισμού (θέλει προσοχή στο όνομα)

Αυτό ΔΕΝ ανεβαίνει με σύρσιμο — φτιάχνεται με το χέρι για να μπει στον σωστό φάκελο:

1. **Add file** → **Create new file**
2. Στο όνομα γράψε ΑΚΡΙΒΩΣ (με τις καθέτους):
   `.github/workflows/scrape.yml`
3. Άνοιξε το αρχείο `scrape.yml` από τον υπολογιστή σου με το Σημειωματάριο,
   αντίγραψε ΟΛΟ το περιεχόμενο και κόλλησέ το στο μεγάλο κουτί
4. Πάτα **Commit changes**

## Βήμα 5 — Πρώτο τρέξιμο της μηχανής

1. Καρτέλα **Actions** (πάνω, δίπλα στο Code)
2. Αν ρωτήσει, πάτα **I understand my workflows, enable them**
3. Αριστερά επίλεξε **Ενημέρωση πλειστηριασμών**
4. Δεξιά πάτα **Run workflow** → πράσινο **Run workflow**
5. Περίμενε 2-5 λεπτά. Πράσινο τικ = τα δεδομένα μαζεύτηκαν.
   Κόκκινο Χ = πάτα πάνω του, άνοιξε το βήμα «Συλλογή δεδομένων»,
   και στείλε στον Claude ό,τι γράφει.

Από εδώ και πέρα τρέχει ΜΟΝΟ ΤΟΥ κάθε πρωί στις 07:00 ώρα Κύπρου.

## Βήμα 6 — Ενεργοποίηση της σελίδας

1. **Settings** (καρτέλα πάνω) → αριστερά **Pages**
2. Στο "Source" επίλεξε **Deploy from a branch**
3. Branch: **main**, φάκελος: **/ (root)** → **Save**
4. Σε 2-3 λεπτά η σελίδα σου ζει στο:
   `https://ΤΟ-USERNAME-ΣΟΥ.github.io/auctions-cy/`

## Έτοιμο. Τι έχεις τώρα

- Σελίδα με όλους τους επερχόμενους πλειστηριασμούς, φίλτρα, αναζήτηση
- Αυτόματη καθημερινή ενημέρωση χωρίς να κάνεις τίποτα
- Ιστορικό τιμών που μεγαλώνει μόνο του (data/history.json) — το ανταγωνιστικό
  σου πλεονέκτημα από την πρώτη μέρα

## Βήμα 7 (προαιρετικό) — Ενεργοποίηση AI ανάλυσης

Χωρίς αυτό το βήμα η σελίδα δουλεύει κανονικά, απλώς χωρίς AI σκορ/περιλήψεις.

1. Πήγαινε στο console.anthropic.com → API Keys → Create Key και αντίγραψέ το
2. Στο GitHub repository: **Settings** → **Secrets and variables** → **Actions**
   → **New repository secret**
3. Όνομα: `ANTHROPIC_API_KEY` — Τιμή: το κλειδί που αντέγραψες → **Add secret**
4. Από το επόμενο τρέξιμο, κάθε ακίνητο παίρνει AI σκορ, αιτιολόγηση,
   προειδοποιήσεις και περίληψη σε 4 γλώσσες. Κόστος: σεντς την ημέρα
   (κάθε ακίνητο αναλύεται μία φορά, ξανά μόνο αν αλλάξει η τιμή του).

## Βήμα 8 (προαιρετικό) — Κουμπιά «Ενδιαφέρομαι» (leads)

1. Πήγαινε στο formspree.io → Sign up (δωρεάν, 50 μηνύματα/μήνα)
2. New Form → δώσε όνομα → αντίγραψε το endpoint (μοιάζει με
   https://formspree.io/f/abcdwxyz)
3. Στο GitHub άνοιξε το index.html → πάτα το μολυβάκι (Edit) →
   βρες τη γραμμή  const LEAD_ENDPOINT = "";  και βάλε το endpoint
   μέσα στα εισαγωγικά → Commit changes
4. Από εκείνη τη στιγμή κάθε ακίνητο έχει κουμπί «Ενδιαφέρομαι» —
   ο επισκέπτης διαλέγει τι θέλει (νομικό έλεγχο, χρηματοδότηση,
   εκτίμηση, συμμετοχή) και το αίτημα έρχεται στο email σου με τον
   κωδικό του ακινήτου. Εσύ απλώς το προωθείς στον συνεργάτη σου.


## SEO — αυτόματες σελίδες (ήδη ενεργό)

Ο αυτοματισμός παράγει μόνος του κάθε μέρα, χωρίς καμία ενέργεια από σένα:
- Μία σελίδα ανά επαρχία (φάκελος `d/`) — π.χ. «Πλειστηριασμοί Πάφου» — που
  πιάνει τις τοπικές αναζητήσεις στη Google, όπως έγινε στο cymechenergy.
- Μία σελίδα ανά πόλη/δήμο (φάκελος `t/`) — π.χ. «Πλειστηριασμοί Παραλιμνίου» —
  που πιάνει αναζητήσεις σε επίπεδο πόλης, όχι μόνο επαρχίας.
- Μία σελίδα ανά ακίνητο (φάκελος `p/`) με δικό της title, περιγραφή, δομημένα
  δεδομένα (schema) και την AI ανάλυση — δεκάδες σελίδες indexable.
- Ενημερωμένο sitemap.xml με όλες αυτές τις σελίδες.

ΣΗΜΑΝΤΙΚΟ για να δουλέψει πλήρως το SEO: μόλις ενεργοποιήσεις το GitHub Pages
και ξέρεις το URL σου, βάλ' το στη γραμμή `SITE_URL` στην αρχή του `scraper.py`
(μέσα από το GitHub, με το μολυβάκι). Τότε ενεργοποιούνται canonical, hreflang
και τα σωστά links στο sitemap. Μετά, στο Google Search Console:
1. Πρόσθεσε το site σου (όπως έκανες για το cymechenergy)
2. Υπόβαλε το `https://ΤΟ-URL-ΣΟΥ/sitemap.xml`

## Βήμα 9 (αργότερα) — Σελίδα «Για επαγγελματίες»

Η σελίδα partners.html είναι ήδη έτοιμη: δικηγόροι, μεσίτες και εκτιμητές
συμπληρώνουν φόρμα για να λαμβάνουν leads αγοραστών — έρχονται σε ΣΕΝΑ,
δεν τους κυνηγάς εσύ. Όσο δεν έχει endpoint, δείχνει «εγγραφές σύντομα».

Ενεργοποίησέ την ΑΦΟΥ έχεις επισκεψιμότητα (τότε οι επαγγελματίες έχουν
λόγο να εγγραφούν):
1. Φτιάξε δεύτερη φόρμα στο formspree.io (ξεχωριστή από τα leads αγοραστών)
2. Στο GitHub άνοιξε partners.html → βάλε το endpoint στη γραμμή
   const PARTNER_ENDPOINT = "";
3. Μοιράσου τη σελίδα σε ομάδες/σελίδες επαγγελματιών ακινήτων

## Βήμα 10 — Ειδοποιήσεις (φέρνει τον κόσμο πίσω)

Το κουμπί «🔔 Ειδοποιήσεις» αφήνει τον επισκέπτη να ζητήσει email όταν βγαίνει
ακίνητο στα κριτήριά του (περιοχή, τιμή, AI σκορ). Έτσι επιστρέφει — και μιλά
σε άλλους. Ενεργοποίηση:
1. Φτιάξε τρίτη φόρμα στο formspree.io (ξεχωριστή)
2. Στο index.html βάλε το endpoint στη γραμμή  const ALERT_ENDPOINT = "";
3. Τα emails που εγγράφονται έρχονται σε σένα. Στην αρχή στέλνεις χειροκίνητα
   μια φορά τη βδομάδα όσα ταιριάζουν (η αναφορά report.html βοηθά). Αργότερα
   αυτοματοποιείται.

## Ενημέρωση επίσημων τιμών (κάθε τρίμηνο, 2 λεπτά)

Στο scraper.py υπάρχει ο πίνακας OFFICIAL_TREND με την επίσημη ετήσια τάση
τιμών ανά επαρχία (από Κεντρική Τράπεζα Κύπρου & Στατιστική Υπηρεσία). Αυτό
δίνει στο AI αξιόπιστη εξωτερική αναφορά για την αγορά.

Κάθε τρίμηνο (προαιρετικά):
1. Άνοιξε centralbank.cy → Residential Property Price Indices, ή cystat.gov.cy → House Price Index
2. Δες την ετήσια μεταβολή % ανά επαρχία
3. Ενημέρωσε τα νούμερα στον πίνακα OFFICIAL_TREND και το OFFICIAL_TREND_ASOF
Δεν είναι scraping — δημόσια νούμερα, τα βάζεις με το χέρι. Νόμιμο και απλό.
